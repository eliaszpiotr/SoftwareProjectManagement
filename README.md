# Software Project Management

Project created for the **Web Applications** course.

## Authors

- Eliasz Piotr
- Tomasz Jankowski

## Project Description

SoftwareProjectManagement is a web application developed in Java using the Spring Boot framework.

The project was created for educational purposes as part of the **Web Applications** course. It uses Maven for project management and includes database support with MySQL and Spring Data JPA.

## Technologies Used

- Java 17
- Spring Boot
- Maven
- Spring Web
- Spring Data JPA
- MySQL

## Main Features

- Web application backend
- Spring Boot project structure
- Database connection with MySQL
- Data persistence using Spring Data JPA
- Maven-based project configuration

## Configuration

The main application configuration is located in:

src/main/resources/application.properties

This file contains settings such as the server port, database connection, and JPA configuration.

## Purpose

The purpose of this project is to practice building a web application using Java and Spring Boot, including basic backend structure, database integration, and application configuration.
